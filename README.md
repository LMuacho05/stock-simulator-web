# Stock Market Simulator (WEB)

This project is a continuation of my initial console-based application:
[https://github.com/LMuacho05/stock-market-simulator](https://github.com/LMuacho05/stock-market-simulator)
-------------------------------------------------------------------------
**Language**: Java (JDK 17+);

**Framework**: Spring Boot

**Database**: MySQL

**External API**: Alpha Vantage (REST)

**Tools**: Maven/Gradle, Git, IntelliJ IDEA

-------------------------------------------------------------------------

### How to run

1. Having MySQL running.
2. Rename config.properties.example to config.properties and change to your API key.
3. Rename application.properties.example to application.properties and change to your DB password.
4. Run DemoApplication.java in your IDE.

### Learning Goals

My main goal is to upgrade my Fullstack development skills. This specific phase of the project is heavily focused on the **Backend**, migrating a standard Java application into a modern Spring Boot REST API, handling HTTP requests, and managing database connections through Spring.
